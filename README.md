# Simple-DS-RichPresence
Un logiciel simple qui permet d'avoir une rich presence discord ( Joue à ... ) personnalisée simplement.

![aperçu](https://niaso.cf/image/SimpleRPC.PNG)

FAQ
Quesqu'une "Application ID" ?
Une Application ID c'est l'id de votre application discord qui va servire a faire votre rich presence. Pour en crée une il faudra aller sur https://discord.com/developers/applications/

![Aide1](https://niaso.cf/image/Aide1.PNG)

Ensuite il faudra cliquer sur "New Application"

![Aide2](https://niaso.cf/image/Aide2.PNG)

Une fois cette pop-up ouverte vous devrez donner un nom a votre application ex : MonAppRPC. Ce nom s'affichera sur votre "joue à ...", Ex : joue à MonAppRPC.
Une fois le nom choisis cliquez sur "Create"

![Aide3](https://niaso.cf/image/Aide3.PNG)

Une fois sur cette page (vous devrez être rediriger automatiquement), vous choisirez une photo pour votre App et vous copierai le "Client ID" en cliquent sur Copy.
Une fois cela fait vous devrez aller dans la categorie "Rich Presence"

![Aide4](https://niaso.cf/image/Aide4.PNG)

 Une fois sur cette page : 
 
 ![Aide5](https://niaso.cf/image/Aide5.PNG)
 
 Vous devrez choisir une image pour votre rich presence. Une fois fait descendez en bas de la page et il y-aura un bouton "Add Image(s) cliquez dessus et ajoutez a nouveaux votre image et optionellement une deuxieme image qui s' affichera en bas a droit de l'image principale et retenez le nom que vous leur avez donner
